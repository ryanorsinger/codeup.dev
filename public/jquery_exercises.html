The real POWER and CORE of jQuery are the Selectors

To use any meta-characters such as !"#$%&'()*+,./:;<=>?@[\]^`{|}~ ) ) as a literal part of a name, it must be escaped with with two backslashes: \\. For example, an element with id="foo.bar", uses the selector $("#foo\\.bar").

List of jQuery Selectors
  • * is the All Selector and selects all elements
  • ID Selector #id Selects a single element with the given id attribute.
  • Class Selector .class Selects all elements with the given class.
  • Element Selector element Selects all elements with the given tag name.
  • Multiple Selector selector1, selector2, selectorN Selects the combined results of all the specified selectors.

    • jQuery manipulates the DOM, select things, change things, wait for things, and do things!

    Multiple Selectors allow us to have even more control over jQuery selections!
